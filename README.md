# Subscription Customer Analytics

## Project Overview

This project analyzes customer subscription behavior for a subscription-based business using SQL. The analysis focuses on customer acquisition, upgrades, churn, and revenue trends to generate business insights and support strategic decision-making.

## Objectives

- Analyze customer subscription behaviour
- Measure customer churn
- Track subscription upgrades and downgrades
- Evaluate customer retention
- Support business decisions using SQL

## Technologies

- SQL
- MySQL
- Git
- GitHub

## SQL Concepts Used

- Joins
- Aggregate Functions
- CTEs
- Window Functions
- CASE Statements
- Date Functions
- GROUP BY
- Subqueries

## Business Questions

- How many customers joined the platform?
- Which subscription plan is the most popular?
- What percentage of customers churned?
- How many customers upgraded to annual plans?
- What is the monthly customer growth?
- Which subscription plans generate the most revenue?
- How does customer behaviour change over time?
